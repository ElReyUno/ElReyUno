Project WeSpark

Developed WeSpark, a comprehensive full-stack "Learning & Opportunity Dashboard" architected within a monorepo to demonstrate end-to-end software development capabilities. The system features three core, integrated components: a cross-platform mobile client for data submission built with .NET MAUI and .NET 7, using C# for logic and XAML for the user interface; a central RESTful backend API powered by Node.js and Express.js, which utilizes the native fs module for initial data persistence to a local JSON file and cors middleware for secure requests; and a sophisticated administrative web dashboard built as an Angular 16 single-page application with TypeScript, HTML5, and CSS3, using the HttpClientModule for API interaction. The entire development ecosystem was managed using Git and GitHub for version control, with a cross-platform workflow spanning Visual Studio 2022 for .NET MAUI and Visual Studio Code within a Windows Subsystem for Linux (WSL) Ubuntu 22.04 LTS environment, all complemented by Postman for API testing and documentation and Canva for design assets. The project was strategically designed with a forward-thinking migration path to a scalable, serverless AWS cloud architecture, leveraging services such as API Gateway, AWS Lambda, Amazon DynamoDB, and IAM for a robust, enterprise-grade deployment

Automate the Deployment of WordPress on Amazon Web Services (AWS)

Engineered and implemented End-to-End DevOps and MLOps solutions on AWS, demonstrating expertise across both traditional web application deployment and modern machine learning lifecycles. Established a robust CI/CD Pipeline for a LAMP/WordPress environment, automating deployments to EC2 within a secure VPC. This solution leveraged Infrastructure as Code (CloudFormation in JSON/YAML) and a comprehensive toolchain including Docker and the AWS DevOps suite, with automation scripted in Bash and Python (boto3). Concurrently, a complete MLOps Pipeline was implemented using AWS SageMaker with GPU instances for accelerated training, orchestrating automated model deployment, monitoring, and retraining through a SageModel Registry, ECR, EventBridge, and SNS. This expertise in cloud-native architecture was further demonstrated by engineering a high-performance, cross-platform application with Flutter, supported by a microservices environment utilizing REST and gRPC APIs. The solution integrated MariaDB on RDS for scalable data persistence and leveraged the AWS CLI for task automation. A comprehensive, multi-layered testing strategy, spanning Front-End (Jest/Cypress), Back-End (PyTest), API (Postman), and performance (JMeter)—validated all solutions, ensuring the delivery of highly reproducible, secure, and cost-optimized cloud environments.

PersonaForge

Engineered and deployed a Full-Stack, AI-powered toolkit architected with Ruby on Rails 8.0, PostgreSQL, and Redis/Sidekiq for robust data storage and background job processing. The system integrates OpenAI's API for intelligent, dynamic generation of professional documents and features secure user authentication with Devise. The entire application is containerized using Docker and deployed to Google Cloud Platform (GCP) resources, including Cloud Run, Cloud SQL, and Memorystore, which are provisioned and managed entirely through Infrastructure as Code (IaC) with modular Terraform configurations. A comprehensive CI/CD Pipeline built with GitHub Actions automates the entire workflow, from running comprehensive RSpec tests and security scans to building and deploying Docker images across multiple environments. Enterprise-grade observability is achieved through an integrated monitoring stack featuring Sentry, Prometheus, and Grafana, providing deep insights into application performance and user behavior. This End-to-End automation and infrastructure management ensures a highly scalable, reliable, and secure platform.
